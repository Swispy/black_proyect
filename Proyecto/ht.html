Html

Nivel 0: Introducción a HTML

Html es un Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

En html el código se representa por medio de etiquetas formando así un elemento en general, las etiquetas tienen etiquetas de aperturas que es donde inicia la etiqueta y la apertura de cierre que es donde finaliza con un/

Nivel 1: Estructura De la pagina Web

se inicia usando dos etiquetas fundamentales, el doctype html y la etiqueta html. el doctype le dirá al navegador que tipo de versión html está usando mientras que la etiqueta html nos indica el inicio de nuestra página web

Nivel 2: Head y Body

La etiqueta head define parte elemental de la página al igual que body, head se encarga de establecer los actos o elementos importantes de la página web mientras que body se encarga de estructurar lo que se va a visualizar en ella

Nivel 3: Etiquetas H y P

Las etiqueta H nos permite generar encabezados de diferentes tamaños esta se nivela de h1 a h6, la etiqueta P nos permite generar un párrafo o un escrito que le queramos dar a la página web

Nivel 4: Title

Como su palabra lo dice title establece un título al texto en la viñeta de arriba

Nivel 5: Imágenes

La etiqueta img nos permite importar una imagen a nuestra página web esto por medio de un atributo llamado src en donde podremos ingresar el link de la imagen o la imagen importada desde nuestro archivador

Nivel 6: Etiqueta Strong Y etiqueta i

Estas Dos etiquetas nos sirven para darle una contextura a la letra, ya sea en negrilla usando Strong o en cursiva con la etiqueta I

Nivel 7: Etiqueta de espaciado Br y Líneas con Hr

Br es aquella etiqueta en la cual podremos generar espaciados entre elementos, mientras más veces pongamos la etiqueta br podremos darnos cuenta que el espaciado se irá haciendo más grande, la etiqueta hr general una línea en la página web reflejándose así como un medio de edición

Nivel 8: Etiquetas Table, td y tr

Las etiquetas Table td y tr nos sirven para generar tablas o columnas en una página web, table se encarga de unir las tablas y filas que queramos crear, td se encarga de crear las columnas mientras que tr se encarga de crear las filas

Nivel 9: Creación de contenedores

en html existen contenedores el cual almacenan elementos relevantes en ella hay dos tipos de contenedores el contenedor div el cual es un contenedor que nos permite guardar en el múltiples contenedores en el y por otro lado tenemos a el cual es un contenedor dependiente la diferencia de estos dos es que section está diseñado para contenidos dependientes, pero diferenciados. No se debe usar el elemento section como un mero contenedor genérico; para esto ya existe div>

nivel 10: vínculos y hipervínculos

Las etiqueta "a" nos permite crear vínculos a otras páginas o a otros lugares de la web esta se constituye por un atributo href el cual nos permite pegar el link a donde queramos ir, mientras por otro lado la etiqueta nav nos permite crear varios vínculos ala misma

Nivel 11: Form, input y labels

Form es una etiqueta de HTML que representa un formulario. En este formulario se agregan los diferentes campos de entrada de datos o de confirmación, así como los botones, que como mínimo ha de haber uno, el de envío, después se daría la etiqueta input en donde esta tiene a su disposición de diversos atributos para su forma si bien para un campo de texto o para un botón en específico

y por último label La etiqueta label permite dar un título o descripción a cada campo del formulario

Nivel 12: Link y script

link y script son atributos o etiquetas que nos permiten vincular la estructura html (página html) con otro tipo de lenguaje, link nos permite vincular la página html con una css u otra distinta esto para que los cambios que hagamos en la página css se vean reflejados en la html y las dos estén unidas

Mientras Tanto script es una etiqueta que nos enlaza la página html con una de java script

Nivel 13: Span

Con la etiqueta podemos personalizar el estilo de solamente una parte del texto

Nivel 14: Lista ordenadas y listas no ordenadas

Las etiquetas OL y LI nos sirven para crear listas, OL para listas ordenadas y UL para listas sin orden. Dentro de las listas, los elementos se identifican con la etiqueta LI.

Nivel 15: Videos en html

Las etiquetas video y source son las que se utilizan para enlazar un video a tu pagina web, después de agregar tu video se le agrega a la etiqueta video un atributo llamado controls para poder reproducir el video
